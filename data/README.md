# Data Directory

Canonical reusable datasets for the approved MNIST latent-DDPM protocol.

Current layout:

- `MNIST/raw/` - raw torchvision MNIST files.
- `MNIST/clean/` - canonical clean MNIST assets.
- `MNIST/noisy_gaussian_v1/` - baseline Gaussian-noisy MNIST assets.

Datasets here are reusable assets, not run-specific experiment outputs.
